## Product Overview

**One-liner:** TikTok Shop 美区选品，从"看英文图表"变成"用中文聊一句"的 AI 选品 Agent。

**What it does:** 用户用自然中文描述类目/价格/增速/竞争度；TrendRadar 在数秒内流式返回 Top 3 推荐 + 综合评分 + 推理链 + 风险点 + 下一步动作；可持续追问，上下文保留；点击任一推荐展开完整商品档案（GMV 趋势、转化率、达人合作、评论情感、1688 供应商）。

**Product category:** TikTok Shop 选品工具 / 跨境电商 AI 助手。目标货架——当用户搜"TikTok Shop 选品工具"、"Kalodata 替代"、"中文 TikTok 选品"时能被看到。

**Product type:** SaaS · Web 应用（Next.js 前端 + FastAPI 后端 + Neon Postgres + DeepSeek LLM）。

**Business model:** Freemium + 新手友好阶梯订阅（**V3 确认**，2026-04-17 via `pricing-strategy`）

| 档位 | 价格 | 定位 | 包含 |
|---|---|---|---|
| **Free** | ¥0 | 试水 | 每日 3 次对话 · 基础竞品分析 · 不含 1688 反搜 |
| **首周体验** | **¥9.9 / 7 天** | Pro 试用钩子 | Pro 全功能（降低新手试错成本） |
| **首月八折** | **¥79 / 月** | Pro 首月续订价 | Pro 全功能（体验期无缝转换） |
| **Pro 月卡** | **¥99 / 月** | 🏆 主推 | 无限对话 + 痛点分析 + 利润测算 + 爆品日报 + 飞书/Notion 导出 |
| **Pro 年卡** | **¥899 / 年** (~¥75/月，省 25%) | 年付优惠 | 同 Pro 月卡 |
| **Team** | **¥399 / 月 · 5 席** | 工作室/团队 | Pro 全功能 + 共享工作区 + 优先客服 |
| **Launch 限时** | 前 100 名预约 = 终身 5 折 | 冷启动钩子 | 严格封顶 100 人 |

**核心锚点：¥99/月 vs Kalodata $99/月（≈¥700）—— 便宜 7 倍是主武器。**

**Pricing Metric:** 按账号订阅（flat per-account）。新手不想算"对话数 × 单价"，越简单越好。Team 档按 seat 数（5 席）。

**Pricing Roadmap（内嵌提价路径）:**
- **T+0 到 T+3 个月**（上线初期）: 价格冻结，只做数据收集——真实转化率 / ARPU / churn / 哪个功能付费率最高
- **T+6 个月**（首批 500–1000 付费用户后）: 首次调整。方案 A 保守 = Pro ¥99→¥129（+30%）/ Team ¥399→¥499；方案 B 激进 = 保留 ¥99 层，新增 "Pro+" ¥199/月（对标 Kalodata Professional）含 API + 白标 + GPT-4 级推理
- **T+12 个月**（扩展 Amazon / 东南亚后）: 按市场分层——Pro US-only ¥99 / Pro US+SEA ¥159 / Pro Global ¥229
- **Grandfather 铁律**: 每次提价老用户锁价 12 个月；每次提价必须同时加功能（"tied to value"），不能裸涨

**风险点对冲:**
- ¥9.9 首周羊毛党 → 绑定手机号 + 新用户定义（首次注册 24h 内）
- 终身 5 折 100 人长期让利约 ¥12 万（= 100 × ¥49.5 × 24 月） → **必须严格封顶 100 人**，提前做好计数系统
- Team ¥399 若 T+3 占比 < 5% → 降到 ¥299 或拆成 ¥199(3 席)+¥399(5 席) 两档

## Target Audience

**Target users:** **TikTok Shop 美区新手卖家**（偏向）
- 零收入 / 月 GMV < 3k 美金的尝试型卖家
- 个体 / 夫妻店 / 2-3 人微团队
- 决策链 = 使用链 = 付费链 = 同一个人
- 本轮定位从"月 GMV 3k–30k 成长期"**下移到"新手"**——landing 文案在后续 copy-editing 里需要相应调整

**Decision-makers:** 店主本人（1 人）。无 B2B 多方决策链条。

**Primary use case:** 新手卖家每天打开 TrendRadar，用一句话让 AI 帮自己**判断"做什么品"**，不再对着英文数据面板发愣、也不再靠刷短视频"感觉选品"。

**Jobs to be done:**
1. **帮我判断一个品值不值得做**（不是只给数据，而是给结论 + 推理）
2. **帮我找到货源**（图搜 1688 同款直接拿报价，算毛利）
3. **帮我拆解竞品痛点**（评论差评归纳 → 反向改款建议）

**Use cases:**
- "给我 3 个美区 30 美金以内、宠物类、增速最快的品"
- "粘贴竞品链接 → AI 归纳 48 条电池续航差吐槽 → 建议改款点"
- "输入成本 → 输出 $24.99 定价 + 58% 毛利 + 1.72 盈亏 ROAS"
- "新手上路：美区家居 30 天起盘模板"

## Problems & Pain Points

**Core problem:** 目前 TikTok Shop 选品工具都是**数据面板**——给你图表，让你自己决策。中国新手卖家**看不懂英文 + 没经验判断 + 负担不起 $99–$399/月的英文订阅**，结果要么凭感觉选品踩坑、要么被付费群 / 代运营割韭菜。

**Why alternatives fall short:**
- **Kalodata / FastMoss / EchoTik**：英文为主（Kalodata、FastMoss）或功能过深（EchoTik），为有经验、月 GMV 十万+ 的品牌方设计，月费 $99+ ≈ 700+ 元
- **嘀嗒狗 / 导航站免费工具**：免费但只给数据不做判断，新手看不懂
- **蝉妈妈 / 抖查查**：国内版抖音，不覆盖 TikTok Shop 海外
- **选品代运营 / 付费社群**：贵、慢、不透明

**What it costs them:**
- 每月被英文工具订阅咬掉 $100-400 × 多工具叠加
- 每天额外花 2-3 小时翻数据仍抓不准方向
- 踩坑一个品 = 几千美金库存积压

**Emotional tension:**
- 信息差焦虑："感觉别人都在赚我在亏"
- 对英文工具的挫败感
- 对国内同行的 FOMO
- 对"AI 会不会瞎推荐"的将信将疑

3. "30 秒挖爆品"是中文工具标配话术 → TrendRadar 的秒级响应只是基本功不是亮点
4. 新手选品共识："小而美、垂直细分、低客单、易运输、纺织/小饰/小家电" → 可写进新手模板

## Differentiation

**Key differentiators:**
1. **中文原生** — 不是翻译，问/答/推理/导出全中文
2. **Agent 而非面板** — 替你判断并给下一步，不是让你看图表自算
3. **新手友好阶梯定价** — 9.9 元首周 → 79 元首月 → 99 元/月，远低于英文工具的 $99+
4. **推理链透明** — 每条推荐都附数据来源和推理过程，不做黑盒
5. **1688 反搜闭环** — 选品 + 货源 + 利润测算在一个流里

**How we do it differently:** SSE 流式 LLM 对话 + 结构化商品卡片 + 供应链反搜 + 评论情感拆解，形成闭环工作流。

**Why that's better:** 新手不需要学习英文工具、不需要自己算 ROAS、不需要翻差评，一句中文问完事。

**Why customers choose us over alternatives:**
- vs Kalodata/FastMoss：便宜 1/7、说中文、帮你判断
- vs 嘀嗒狗：免费的只给数据，我们给结论
- vs EchoTik：更垂直（只做选品不做全链路）、更便宜、更简单

### Top Themes（按 频率 × 情绪强度 排序）

**Theme 1 · "工具太贵，新手玩不起"**
- 频率：6/6 来源都出现
- 情绪强度：🔥 高（"离谱"、"劝退"、"用不起"）
- 代表原话:
  - "Kalodata useful but expensive for sellers still testing their first product" — Reddit 跨境 seller 吐槽
  - "Kalodata starter $45.9/mo, professional $99/mo, FastMoss basic $59 还锁 revenue data" — Reddit 对比贴
  - "EchoTik $9.90/mo, 75% cheaper than Kalodata" — 被反复当成替代品提及
  - "还没出第一单就先交 700 多块工具费，真的会怀疑人生" — 小红书新手贴
- 含义：**99 元/月 + 9.9 元首周**是强感知价格锚点；落地页应直接对比"Kalodata $99 = 700+ 元"

**Theme 2 · "踩过坑，亏了几万，才明白选品是门玄学"**
- 频率：4/6 来源
- 情绪强度：🔥 高（"亏"、"积压"、"怀疑人生"、"2 万块学费"）
- 代表原话:
  - "踩了 2 万块坑后，做 TikTok 第 3 个月，我终于第一次没亏钱" — 知乎长贴
  - "从 Amazon 转 TikTok，原来的 20 万库存全积压在仓库" — 跨境社群
  - "凭感觉选品，以为抖音爆的在 TikTok 也会爆，结果一单没出" — 小红书
  - "跟着老师清单下单，3000 刀货压手里，老师说'市场变了'" — 知乎
- 含义：**踩坑焦虑 > 数据焦虑**。落地页的钩子应该是"不再凭感觉 / 不再跟单老师"，而不是"更多数据"

**Theme 3 · "英文工具看不懂，每天花 2–3 小时还是判断不出来"**

**Theme 4 · "美区竞争已经定型，新手要挑小而美的缝"**

- 更新频率：每日
